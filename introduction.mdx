# Welcome to UNC

UNC (Universal Neural Compiler) compiles HuggingFace transformer models into optimised native Metal inference binaries. No runtime framework, no Python — just a compiled binary that runs your model at near-hardware-limit speed on Apple Silicon.

## Key Features

- **Compile to Native**: Transform HuggingFace models into standalone Metal binaries
- **Multiple Quantization Levels**: F16, Q8_0, Q4_0 for size/speed tradeoffs
- **Hardware-Agnostic IR**: Clean intermediate representation with Metal backend (CUDA/ROCm/WASM planned)
- **Advanced Optimization**: Kernel fusion, memory planning, dual-path GEMM/GEMV
- **Broad Model Support**: LLaMA, Mistral, Qwen, Phi, Gemma architectures
- **Two Output Modes**: JIT (.unc bundles) for development, AOT (standalone binaries) for production

## How It Works

```
HuggingFace model
       |
  [ Frontend ]        Parse config.json + safetensors
       |
  [ IR Graph ]        Hardware-agnostic tensor graph
       |
  [ Compiler ]        Fusion, quantization, memory planning
       |
       +------------------+------------------+
       |                  |                  |
  [ Metal ]          [ CUDA ]          [ ROCm ]
  Obj-C + Metal      (planned)         (planned)
  shaders
       |
  Native binary
```

The compiler parses your HuggingFace model, builds a typed tensor IR graph, applies optimization passes (fusion, quantization, memory planning), and emits either a JIT-compiled `.unc` bundle or an AOT standalone binary.

## Quick Example

```bash
# Compile TinyLlama to a .unc bundle with Q4_0 quantization
unc compile --model TinyLlama/TinyLlama-1.1B-Chat-v1.0 --quant q4-0 -o ./tinyllama

# Run inference
unc run ./tinyllama.unc --prompt "The history of" --max-tokens 200
```
