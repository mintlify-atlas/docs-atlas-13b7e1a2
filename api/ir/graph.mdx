# IR Graph

CompIR: The computation graph. A DAG of `Node`s where every node has a concrete op, fully-resolved input/output shapes, and (after the kernel matching pass) an assigned kernel.

## Node Representation

### Node Struct

A single operation in the computation graph.

```rust
pub struct Node {
    pub id: NodeId,
    pub op: Op,
    pub inputs: Vec<TensorId>,
    pub output: TensorId,
    pub extra_outputs: Vec<TensorId>,
    pub attrs: AttrMap,
    pub kernel: Option<KernelAssignment>,
    pub estimated_cost: Option<f64>,
}
```

#### Fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | `NodeId` | Unique identifier for this node |
| `op` | `Op` | The operation type (MatMul, RMSNorm, etc.) |
| `inputs` | `Vec<TensorId>` | Input tensor IDs |
| `output` | `TensorId` | Primary output tensor ID |
| `extra_outputs` | `Vec<TensorId>` | Additional outputs (e.g., Split produces multiple outputs) |
| `attrs` | `AttrMap` | Op-specific attributes |
| `kernel` | `Option<KernelAssignment>` | Assigned kernel after matching pass |
| `estimated_cost` | `Option<f64>` | Estimated execution cost in microseconds |

**Example:**
```rust
let node = Node {
    id: NodeId(0),
    op: Op::RMSNorm { eps: 1e-5 },
    inputs: vec![TensorId(0), TensorId(1)],  // input, weight
    output: TensorId(2),
    extra_outputs: vec![],
    attrs: HashMap::new(),
    kernel: None,  // Assigned later by kernel matching
    estimated_cost: None,
};
```

### KernelAssignment Struct

A kernel assigned to a node after the matching pass.

```rust
pub struct KernelAssignment {
    pub kernel_id: KernelId,
    pub kernel_name: String,
    pub dispatch: DispatchConfig,
    pub path: ExecutionPath,
}
```

**Example:**
```rust
let assignment = KernelAssignment {
    kernel_id: KernelId(42),
    kernel_name: "rms_norm_f16".to_string(),
    dispatch: DispatchConfig {
        grid: [
            DispatchDim::CeilDiv { param: ParamName::SeqLen, divisor: 256 },
            DispatchDim::Static(1),
            DispatchDim::Static(1),
        ],
        threadgroup: [256, 1, 1],
        shared_memory_bytes: 0,
    },
    path: ExecutionPath::Unified,
};
```

### ExecutionPath Enum

Execution path for ops that have prefill vs decode variants.

```rust
pub enum ExecutionPath {
    Prefill,
    Decode,
    Unified,
    Dual {
        prefill_kernel: KernelId,
        prefill_kernel_name: String,
        prefill_dispatch: Box<DispatchConfig>,
        decode_kernel: KernelId,
        decode_kernel_name: String,
        decode_dispatch: Box<DispatchConfig>,
    },
}
```

| Variant | Description |
|---------|-------------|
| `Prefill` | Only used during prefill (seq_len > 1) |
| `Decode` | Only used during decode (seq_len = 1) |
| `Unified` | Single kernel for both prefill and decode |
| `Dual` | Separate kernels for prefill and decode |

**Example:**
```rust
// Unified kernel (RMSNorm, activations)
let unified = ExecutionPath::Unified;

// Decode-only kernel (MatVec)
let decode_only = ExecutionPath::Decode;

// Dual path (MatMul for prefill, MatVec for decode)
let dual = ExecutionPath::Dual {
    prefill_kernel: KernelId(10),
    prefill_kernel_name: "matmul_f16_prefill".to_string(),
    prefill_dispatch: Box::new(DispatchConfig {
        grid: [
            DispatchDim::CeilDiv { param: ParamName::SeqLen, divisor: 64 },
            DispatchDim::Static(64),
            DispatchDim::Static(1),
        ],
        threadgroup: [64, 4, 1],
        shared_memory_bytes: 8192,
    }),
    decode_kernel: KernelId(11),
    decode_kernel_name: "matvec_f16_decode".to_string(),
    decode_dispatch: Box::new(DispatchConfig {
        grid: [
            DispatchDim::Static(64),
            DispatchDim::Static(1),
            DispatchDim::Static(1),
        ],
        threadgroup: [256, 1, 1],
        shared_memory_bytes: 0,
    }),
};
```

## Dispatch Configuration

### DispatchConfig Struct

GPU dispatch configuration.

```rust
pub struct DispatchConfig {
    pub grid: [DispatchDim; 3],
    pub threadgroup: [u32; 3],
    pub shared_memory_bytes: u32,
}
```

**Example:**
```rust
let dispatch = DispatchConfig {
    grid: [
        DispatchDim::CeilDiv { param: ParamName::SeqLen, divisor: 256 },
        DispatchDim::Static(32),
        DispatchDim::Static(1),
    ],
    threadgroup: [256, 1, 1],
    shared_memory_bytes: 4096,
};
```

### DispatchDim Enum

A dispatch dimension that may depend on runtime parameters.

```rust
pub enum DispatchDim {
    Static(u32),
    CeilDiv { param: ParamName, divisor: u32 },
    Expr(ParamExpr),
}
```

| Variant | Description | Example |
|---------|-------------|----------|
| `Static(u32)` | Fixed grid dimension | `Static(64)` |
| `CeilDiv { param, divisor }` | Ceiling division of runtime param | `CeilDiv { param: SeqLen, divisor: 256 }` → ⌈seq_len / 256⌉ |
| `Expr(ParamExpr)` | Complex runtime expression | See `ParamExpr` below |

**Example:**
```rust
// Static: 32 thread groups
let static_dim = DispatchDim::Static(32);

// Dynamic: ceil(seq_len / 256) thread groups
let dynamic_dim = DispatchDim::CeilDiv {
    param: ParamName::SeqLen,
    divisor: 256,
};

// Complex expression
let expr_dim = DispatchDim::Expr(ParamExpr::CeilDiv(
    Box::new(ParamExpr::Mul(
        Box::new(ParamExpr::Param(ParamName::SeqLen)),
        Box::new(ParamExpr::Const(32)),
    )),
    Box::new(ParamExpr::Const(256)),
));
```

### ParamExpr Enum

Simple expression for runtime-computed dispatch dimensions.

```rust
pub enum ParamExpr {
    Param(ParamName),
    Const(u32),
    CeilDiv(Box<ParamExpr>, Box<ParamExpr>),
    Mul(Box<ParamExpr>, Box<ParamExpr>),
    Add(Box<ParamExpr>, Box<ParamExpr>),
    Min(Box<ParamExpr>, Box<ParamExpr>),
}
```

**Example:**
```rust
// (seq_len + 255) / 256
let expr = ParamExpr::CeilDiv(
    Box::new(ParamExpr::Add(
        Box::new(ParamExpr::Param(ParamName::SeqLen)),
        Box::new(ParamExpr::Const(255)),
    )),
    Box::new(ParamExpr::Const(256)),
);

// min(seq_len, 8192)
let bounded = ParamExpr::Min(
    Box::new(ParamExpr::Param(ParamName::SeqLen)),
    Box::new(ParamExpr::Const(8192)),
);
```

## Graph Structure

### CompGraph Struct

The full computation graph for a model.

```rust
pub struct CompGraph {
    pub nodes: Vec<Node>,
    pub node_map: HashMap<NodeId, usize>,
    pub tensors: HashMap<TensorId, TensorRef>,
    pub inputs: Vec<TensorId>,
    pub outputs: Vec<TensorId>,
    pub metadata: GraphMetadata,

    next_node_id: u32,
    next_tensor_id: u32,
}
```

#### Fields

| Field | Type | Description |
|-------|------|-------------|
| `nodes` | `Vec<Node>` | All nodes in topological order |
| `node_map` | `HashMap<NodeId, usize>` | Map from NodeId to index in `nodes` vector |
| `tensors` | `HashMap<TensorId, TensorRef>` | All tensors in the graph |
| `inputs` | `Vec<TensorId>` | Graph input tensors (e.g., input_ids, position_ids) |
| `outputs` | `Vec<TensorId>` | Graph output tensors (e.g., logits) |
| `metadata` | `GraphMetadata` | Model metadata |
| `next_node_id` | `u32` | Next available node ID (private) |
| `next_tensor_id` | `u32` | Next available tensor ID (private) |

## Graph Metadata

### GraphMetadata Struct

Metadata about the compiled model.

```rust
pub struct GraphMetadata {
    pub model_id: String,
    pub architecture: ArchitectureFamily,
    pub params: ModelParams,
    pub weight_files: Vec<String>,
    pub total_weight_bytes: usize,
    pub tokenizer_path: Option<String>,
    pub bos_token_id: Option<u32>,
    pub eos_token_id: Option<u32>,
    pub weight_file_paths: Vec<String>,
    pub quant: String,
}
```

#### Fields

| Field | Type | Description |
|-------|------|-------------|
| `model_id` | `String` | HuggingFace model identifier |
| `architecture` | `ArchitectureFamily` | Model architecture family |
| `params` | `ModelParams` | Model hyperparameters |
| `weight_files` | `Vec<String>` | Weight file names |
| `total_weight_bytes` | `usize` | Total size of all weights |
| `tokenizer_path` | `Option<String>` | Absolute path to tokenizer.json in HF cache |
| `bos_token_id` | `Option<u32>` | BOS token id (1 for LLaMA/Mistral) |
| `eos_token_id` | `Option<u32>` | EOS token id (generation stops at this token) |
| `weight_file_paths` | `Vec<String>` | Absolute paths to .safetensors shards for mmap |
| `quant` | `String` | Weight quantization format ("f16", "q8_0", "q4_0") |

**Example:**
```rust
let metadata = GraphMetadata {
    model_id: "meta-llama/Llama-3-8B".to_string(),
    architecture: ArchitectureFamily::LLaMA,
    params: ModelParams { /* ... */ },
    weight_files: vec!["model.safetensors".to_string()],
    total_weight_bytes: 16_000_000_000,
    tokenizer_path: Some("/home/.cache/huggingface/hub/.../tokenizer.json".to_string()),
    bos_token_id: Some(1),
    eos_token_id: Some(2),
    weight_file_paths: vec!["/path/to/model.safetensors".to_string()],
    quant: "f16".to_string(),
};
```

### AttentionType Enum

Attention mechanism variant.

```rust

    MHA,  // Multi-Head Attention
```

| Variant | Description | num_kv_heads |
|---------|-------------|---------------|
| `MHA` | Multi-Head Attention | = num_attention_heads |
| `GQA` | Grouped-Query Attention | < num_attention_heads |
| `MQA` | Multi-Query Attention | = 1 |

```rust
    eos_token_id: Some(2),
    weight_file_paths: vec![],
    quant: "f16".to_string(),
```

### new_tensor()

Create a new tensor and register it in the graph.

```rust
pub fn new_tensor(
    &mut self,
    shape: Shape,
    dtype: DType,
    storage: StorageClass
) -> TensorId
```

**Example:**
```rust

let input_tensor = graph.new_tensor(

        Dim::Static(1),

            min: 1,
            max: 8192,
            alignment: 1,

        Dim::Static(4096),

    DType::F16,
```

## Real-World Examples

### Building a Simple Graph

```rust
use std::collections::HashMap;

// Create graph
let metadata = GraphMetadata {
    model_id: "test-model".to_string(),
    architecture: ArchitectureFamily::LLaMA,
    params: ModelParams { /* ... */ },
    weight_files: vec![],
    total_weight_bytes: 0,
    tokenizer_path: None,
    bos_token_id: Some(1),
    eos_token_id: Some(2),
    weight_file_paths: vec![],
    quant: "f16".to_string(),
};

let mut graph = CompGraph::new(metadata);

// Create tensors
let input = graph.new_tensor(
    Shape(vec![Dim::Static(1), Dim::Static(128), Dim::Static(4096)]),
    DType::F16,
    StorageClass::Activation { buffer: None },
);

let weight = graph.new_tensor(
    Shape(vec![Dim::Static(4096)]),
    DType::F16,
    StorageClass::Weight(WeightBinding {
        name: "norm.weight".to_string(),
        byte_offset: 0,
        byte_size: 4096 * 2,
        file_shape: vec![4096],
        file_dtype: DType::F16,
    }),
);

let output = graph.new_tensor(
    Shape(vec![Dim::Static(1), Dim::Static(128), Dim::Static(4096)]),
    DType::F16,
    StorageClass::Activation { buffer: None },
);

// Add node
let node_id = graph.add_node(
    Op::RMSNorm { eps: 1e-5 },
    vec![input, weight],
    output,
    HashMap::new(),
);

// Set graph inputs/outputs
graph.inputs = vec![input];
graph.outputs = vec![output];

println!("Graph has {} nodes", graph.num_nodes());
```

### Analyzing Graph Statistics

```rust
// Count operation types
let mut op_counts = HashMap::new();
for node in graph.topo_iter() {
    let op_name = format!("{:?}", node.op);
    *op_counts.entry(op_name).or_insert(0) += 1;
}

println!("Operation counts:");
for (op, count) in op_counts {
    println!("  {}: {}", op, count);
}

// Compute performance estimate
let total_cost: f64 = graph.topo_iter()
    .filter_map(|n| n.estimated_cost)
    .sum();
println!("Estimated total cost: {:.2} ms", total_cost / 1000.0);

// Analyze memory usage
let total_activations: usize = graph.tensors.values()
    .filter(|t| matches!(t.storage, StorageClass::Activation { .. }))
    .map(|t| t.max_size_bytes())
    .sum();
println!("Peak activation memory: {} MB", total_activations / (1024 * 1024));
```

### Walking the Graph

```rust
// Print graph in execution order
for (idx, node) in graph.topo_iter().enumerate() {
    println!("Step {}: {:?}", idx, node.op);
    
    // Print inputs
    for input_id in &node.inputs {
        let tensor = graph.tensor(*input_id);
        println!("  Input: {} {} bytes",
            tensor.shape,
            tensor.max_size_bytes(),
        );
    }
    
    // Print output
    let output = graph.tensor(node.output);
    println!("  Output: {} {} bytes",
        output.shape,
        output.max_size_bytes(),
    );
    
    // Print kernel assignment
    if let Some(kernel) = &node.kernel {
        println!("  Kernel: {}", kernel.kernel_name);
    }
}
```
