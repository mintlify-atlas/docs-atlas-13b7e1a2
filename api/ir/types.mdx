# IR Type System

Core type system for the UNC IR. Every tensor in the graph has a fully-resolved shape, dtype, and storage class.

<Note>
**Key invariant:** Weight dimensions are always `Dim::Static` (known at compile time), while sequence/batch dimensions are `Dim::Param` (known at runtime, but bounded).
</Note>

## Identifiers

Unique identifiers for graph elements.

### NodeId

Unique identifier for a node in the computation graph.

```rust
pub struct NodeId(pub u32);
```

### TensorId

Unique identifier for a tensor (output of a node or a weight).

```rust
pub struct TensorId(pub u32);
```

### BufferId

Unique identifier for a buffer in the memory plan.

```rust
pub struct BufferId(pub u32);
```

### KernelId

Unique identifier for a kernel in the registry.

```rust
pub struct KernelId(pub u32);
```

## Dimension Expressions

### Dim Enum

A dimension that is either known at compile time or parameterized.

```rust
pub enum Dim {
    /// Fully resolved at compile time (e.g., hidden_dim = 4096)
    Static(usize),
    /// Resolved at runtime but bounded (e.g., seq_len ∈ [1, 8192])
    Param(ParamDim),
}
```

Weight dimensions are always `Static`. Sequence length and batch size are `Param` with known upper bounds — the compiler allocates for `max` and dispatches for actual.

#### Methods

| Method | Signature | Description |
|--------|-----------|-------------|
| `max_value()` | `pub fn max_value(&self) -> usize` | Returns the compile-time value, or the max bound for parameterized dims |
| `min_value()` | `pub fn min_value(&self) -> usize` | Returns the compile-time value, or the min bound for parameterized dims |
| `as_static()` | `pub fn as_static(&self) -> Option<usize>` | Returns `Some(value)` if this dimension is statically known |
| `is_static()` | `pub fn is_static(&self) -> bool` | Returns `true` if this dimension is static |

**Example usage:**
```rust
// Static dimension (weight dimension)
let hidden_dim = Dim::Static(4096);
assert_eq!(hidden_dim.max_value(), 4096);
assert_eq!(hidden_dim.as_static(), Some(4096));
assert!(hidden_dim.is_static());

// Parameterized dimension (sequence length)
let seq_len = Dim::Param(ParamDim {
    name: ParamName::SeqLen,
    min: 1,
    max: 8192,
    alignment: 1,
});
assert_eq!(seq_len.max_value(), 8192);
assert_eq!(seq_len.min_value(), 1);
assert_eq!(seq_len.as_static(), None);
assert!(!seq_len.is_static());
```

### ParamDim

A runtime-parameterized dimension with constraints.

```rust
pub struct ParamDim {
    pub name: ParamName,
    pub min: usize,
    pub max: usize,
    /// Required alignment (e.g., 8 for SIMD). 1 means no alignment constraint.
    pub alignment: usize,
}
```

**Example:**
```rust
// Sequence length: [1, 8192], no alignment requirement
let seq_len = ParamDim {
    name: ParamName::SeqLen,
    min: 1,
    max: 8192,
    alignment: 1,
};

// Batch size: [1, 32], must be multiple of 4 for SIMD
let batch_size = ParamDim {
    name: ParamName::BatchSize,
    min: 1,
    max: 32,
    alignment: 4,
};
```

### ParamName Enum

Named runtime parameters.

```rust
pub enum ParamName {
    SeqLen,
    TotalSeqLen,  // seq_len + kv_cache_len (for attention)
    BatchSize,
}
```

| Variant | Display | Description |
|---------|---------|-------------|
| `SeqLen` | `"seq"` | Input sequence length (new tokens) |
| `TotalSeqLen` | `"total_seq"` | Total sequence length including KV cache |
| `BatchSize` | `"batch"` | Batch size |

**Example:**
```rust
use std::fmt::Write;

let mut s = String::new();
write!(&mut s, "{}", ParamName::SeqLen).unwrap();
assert_eq!(s, "seq");
```

## Shape

### Shape Struct

A tensor shape where each dimension is either static or parameterized.

```rust
pub struct Shape(pub Vec<Dim>);
```

#### Methods

| Method | Signature | Description |
|--------|-----------|-------------|
| `rank()` | `pub fn rank(&self) -> usize` | Number of dimensions |
| `is_fully_static()` | `pub fn is_fully_static(&self) -> bool` | Returns `true` if all dimensions are statically known |
| `max_shape()` | `pub fn max_shape(&self) -> Vec<usize>` | Returns the shape with all param dims replaced by their max bounds |
| `max_numel()` | `pub fn max_numel(&self) -> usize` | Total number of elements at maximum dimensions |

**Example usage:**
```rust
// Weight tensor: [4096, 4096] - fully static
let weight_shape = Shape(vec![
    Dim::Static(4096),
    Dim::Static(4096),
]);
assert_eq!(weight_shape.rank(), 2);
assert!(weight_shape.is_fully_static());
assert_eq!(weight_shape.max_shape(), vec![4096, 4096]);
assert_eq!(weight_shape.max_numel(), 4096 * 4096);

// Activation tensor: [batch, seq, hidden] - dynamic batch and seq
let activation_shape = Shape(vec![
    Dim::Param(ParamDim {
        name: ParamName::BatchSize,
        min: 1,
        max: 32,
        alignment: 1,
    }),
    Dim::Param(ParamDim {
        name: ParamName::SeqLen,
        min: 1,
        max: 8192,
        alignment: 1,
    }),
    Dim::Static(4096),
]);
assert_eq!(activation_shape.rank(), 3);
assert!(!activation_shape.is_fully_static());
assert_eq!(activation_shape.max_shape(), vec![32, 8192, 4096]);
assert_eq!(activation_shape.max_numel(), 32 * 8192 * 4096);

// Display format
let s = format!("{}", activation_shape);
assert_eq!(s, "[batch, seq, 4096]");
```

## Data Types

### DType Enum

Tensor data types, including quantized formats.

```rust
pub enum DType {
    F32,
    F16,
    BF16,
    // Quantized types (block-structured)
    Q8_0,   // 8-bit: 32 elements per block, 1 x f16 scale
    Q4_0,   // 4-bit: 32 elements per block, 1 x f16 scale (symmetric)
    Q4_1,   // 4-bit: 32 elements per block, f16 scale + f16 min
    // Integer types (for indices, masks)
    I32,
    U32,
    Bool,
}
```

#### Floating Point Types

| Type | Size | Description |
|------|------|-------------|
| `F32` | 4 bytes | 32-bit floating point |
| `F16` | 2 bytes | 16-bit floating point (IEEE 754) |
| `BF16` | 2 bytes | Brain float 16 (truncated F32) |

#### Quantized Types

Block-structured quantization formats for compressed weights.

| Type | Block Size | Elements/Block | Description |
|------|-----------|----------------|-------------|
| `Q8_0` | 34 bytes | 32 | 8-bit symmetric: 32 × i8 + 1 × f16 scale |
| `Q4_0` | 18 bytes | 32 | 4-bit symmetric: 16 bytes data + 2 bytes scale |
| `Q4_1` | 20 bytes | 32 | 4-bit asymmetric: 16 bytes data + 2 bytes scale + 2 bytes min |

#### Integer Types

| Type | Size | Description |
|------|------|-------------|
| `I32` | 4 bytes | 32-bit signed integer |
| `U32` | 4 bytes | 32-bit unsigned integer |
| `Bool` | 1 byte | Boolean (0 or 1) |

#### Methods

| Method | Signature | Description |
|--------|-----------|-------------|
| `element_size()` | `pub fn element_size(&self) -> Option<usize>` | Size in bytes per element (for non-quantized types) |
| `block_num_elements()` | `pub fn block_num_elements(&self) -> Option<usize>` | For quantized types: number of elements per quantization block |
| `block_size_bytes()` | `pub fn block_size_bytes(&self) -> Option<usize>` | For quantized types: size in bytes per block |
| `is_quantized()` | `pub fn is_quantized(&self) -> bool` | Returns `true` if this is a quantized type |
| `accum_dtype()` | `pub fn accum_dtype(&self) -> DType` | The accumulation dtype for compute involving this type |

**Example usage:**
```rust
// Floating point types
let f32_dtype = DType::F32;
assert_eq!(f32_dtype.element_size(), Some(4));
assert!(!f32_dtype.is_quantized());
assert_eq!(f32_dtype.accum_dtype(), DType::F32);

let f16_dtype = DType::F16;
assert_eq!(f16_dtype.element_size(), Some(2));

// Quantized types
let q4_dtype = DType::Q4_0;
assert_eq!(q4_dtype.element_size(), None);
assert_eq!(q4_dtype.block_num_elements(), Some(32));
assert_eq!(q4_dtype.block_size_bytes(), Some(18));
assert!(q4_dtype.is_quantized());
assert_eq!(q4_dtype.accum_dtype(), DType::F32);

let q8_dtype = DType::Q8_0;
assert_eq!(q8_dtype.block_num_elements(), Some(32));
assert_eq!(q8_dtype.block_size_bytes(), Some(34));

// Integer types
let i32_dtype = DType::I32;
assert_eq!(i32_dtype.element_size(), Some(4));
assert!(!i32_dtype.is_quantized());

// Display format
assert_eq!(format!("{}", DType::F32), "f32");
assert_eq!(format!("{}", DType::Q4_0), "q4_0");
```

## Storage Classes

### StorageClass Enum

Where a tensor's data lives and how it's accessed.

```rust
pub enum StorageClass {
    /// Model weight: mmap'd from safetensors/GGUF at a known byte offset
    Weight(WeightBinding),

    /// Activation: computed at runtime, stored in a reusable buffer
    Activation {
        buffer: Option<BufferId>,
    },

    /// KV cache: persistent across decode steps, pre-allocated for max_seq_len
    KVCache {
        layer: usize,
        head_group: usize,
        role: KVRole,
    },

    /// Compile-time constant (e.g., RoPE frequency table, causal mask template)
    Constant {
        data: Vec<u8>,
    },
}
```

| Variant | Description | Use Case |
|---------|-------------|----------|
| `Weight(WeightBinding)` | Model weight mmap'd from file | All model parameters |
| `Activation { buffer }` | Runtime-computed activation | Intermediate results |
| `KVCache { layer, head_group, role }` | Persistent KV cache | Attention key/value history |
| `Constant { data }` | Compile-time constant | RoPE freqs, masks |

**Example usage:**
```rust
// Weight tensor
let weight_storage = StorageClass::Weight(WeightBinding {
    name: "model.layers.0.self_attn.q_proj.weight".to_string(),
    byte_offset: 0,
    byte_size: 4096 * 4096 * 2,
    file_shape: vec![4096, 4096],
    file_dtype: DType::F16,
});

// Activation tensor (buffer assigned later)
let activation_storage = StorageClass::Activation { buffer: None };

// KV cache for layer 0 keys
let kv_storage = StorageClass::KVCache {
    layer: 0,
    head_group: 0,
    role: KVRole::Key,
};

// Constant (RoPE frequency table)
let const_storage = StorageClass::Constant {
    data: vec![0u8; 1024],
};
```

### KVRole Enum

Whether a KV cache tensor holds keys or values.

```rust
pub enum KVRole {
    Key,
    Value,
}
```

**Example:**
```rust
let key_cache = StorageClass::KVCache {
    layer: 0,
    head_group: 0,
    role: KVRole::Key,
};

let value_cache = StorageClass::KVCache {
    layer: 0,
    head_group: 0,
    role: KVRole::Value,
};
```

### WeightBinding Struct

Binding a tensor to a specific location in the weight file.

```rust
pub struct WeightBinding {
    /// Name in the safetensors file
    pub name: String,
    /// Byte offset from the start of the weight data section
    pub byte_offset: usize,
    /// Size in bytes
    pub byte_size: usize,
    /// Original shape as stored in the file
    pub file_shape: Vec<usize>,
    /// Original dtype as stored in the file
    pub file_dtype: DType,
}
```

**Example:**
```rust
let binding = WeightBinding {
    name: "model.layers.0.self_attn.q_proj.weight".to_string(),
    byte_offset: 12345678,
    byte_size: 33554432,  // 4096 * 4096 * 2 bytes (f16)
    file_shape: vec![4096, 4096],
    file_dtype: DType::F16,
};
```

## Tensor Reference

### TensorRef Struct

A fully-described tensor in the computation graph.

```rust
pub struct TensorRef {
    pub id: TensorId,
    pub shape: Shape,
    pub dtype: DType,
    pub storage: StorageClass,
    /// Memory layout: stride per dimension in elements.
    /// None means default row-major (C-contiguous).
    pub strides: Option<Vec<usize>>,
}
```

#### Methods

| Method | Signature | Description |
|--------|-----------|-------------|
| `max_size_bytes()` | `pub fn max_size_bytes(&self) -> usize` | Total size in bytes at maximum dimensions |
| `decode_size_bytes()` | `pub fn decode_size_bytes(&self) -> usize` | Size in bytes at decode dimensions (Param dims use min value, i.e. seq_len=1) |

**Example usage:**
```rust
// Create activation tensor: [batch, seq, hidden]
let tensor = TensorRef {
    id: TensorId(0),
    shape: Shape(vec![
        Dim::Param(ParamDim {
            name: ParamName::BatchSize,
            min: 1,
            max: 32,
            alignment: 1,
        }),
        Dim::Param(ParamDim {
            name: ParamName::SeqLen,
            min: 1,
            max: 8192,
            alignment: 1,
        }),
        Dim::Static(4096),
    ]),
    dtype: DType::F16,
    storage: StorageClass::Activation { buffer: None },
    strides: None,  // Row-major
};

// Allocate buffer for maximum size (prefill: batch=32, seq=8192)
let max_bytes = tensor.max_size_bytes();
assert_eq!(max_bytes, 32 * 8192 * 4096 * 2);

// Decode uses minimal size (batch=1, seq=1)
let decode_bytes = tensor.decode_size_bytes();
assert_eq!(decode_bytes, 1 * 1 * 4096 * 2);
```

### Quantized Tensor Example

```rust
// Quantized weight tensor: [4096, 4096] in Q4_0 format
let q_tensor = TensorRef {
    id: TensorId(1),
    shape: Shape(vec![
        Dim::Static(4096),
        Dim::Static(4096),
    ]),
    dtype: DType::Q4_0,
    storage: StorageClass::Weight(WeightBinding {
        name: "q_proj.weight".to_string(),
        byte_offset: 0,
        byte_size: (4096 * 4096 / 32) * 18,  // Block-structured
        file_shape: vec![4096, 4096],
        file_dtype: DType::Q4_0,
    }),
    strides: None,
};

let numel = 4096 * 4096;
let block_elems = q_tensor.dtype.block_num_elements().unwrap();
let block_bytes = q_tensor.dtype.block_size_bytes().unwrap();
let total_bytes = (numel / block_elems) * block_bytes;
assert_eq!(q_tensor.max_size_bytes(), total_bytes);
```

## Attributes

### Attr Enum

Op-specific attributes carried on graph nodes.

```rust
pub enum Attr {
    Int(i64),
    Float(f64),
    String(String),
    Ints(Vec<i64>),
    Floats(Vec<f64>),
    Bool(bool),
}
```

### AttrMap Type

Convenience type for the attribute map.

```rust
pub type AttrMap = HashMap<String, Attr>;
```

**Example usage:**
```rust
use std::collections::HashMap;

let mut attrs: AttrMap = HashMap::new();
attrs.insert("eps".to_string(), Attr::Float(1e-5));
attrs.insert("axis".to_string(), Attr::Int(-1));
attrs.insert("transpose_b".to_string(), Attr::Bool(true));
attrs.insert("perm".to_string(), Attr::Ints(vec![0, 2, 1, 3]));
```

## Real-World Examples

### Creating a Weight Tensor

```rust
// Q-projection weight for attention: [4096, 4096] in f16
let q_weight = TensorRef {
    id: TensorId(10),
    shape: Shape(vec![
        Dim::Static(4096),
        Dim::Static(4096),
    ]),
    dtype: DType::F16,
    storage: StorageClass::Weight(WeightBinding {
        name: "model.layers.0.self_attn.q_proj.weight".to_string(),
        byte_offset: 0,
        byte_size: 4096 * 4096 * 2,
        file_shape: vec![4096, 4096],
        file_dtype: DType::F16,
    }),
    strides: None,
};
```

### Creating an Activation Tensor

```rust
// Hidden state: [batch, seq, 4096]
let hidden = TensorRef {
    id: TensorId(20),
    shape: Shape(vec![
        Dim::Param(ParamDim {
            name: ParamName::BatchSize,
            min: 1,
            max: 1,  // Single sequence generation
            alignment: 1,
        }),
        Dim::Param(ParamDim {
            name: ParamName::SeqLen,
            min: 1,
            max: 8192,
            alignment: 1,
        }),
        Dim::Static(4096),
    ]),
    dtype: DType::F16,
    storage: StorageClass::Activation { buffer: None },
    strides: None,
};
```

### Creating a KV Cache Tensor

```rust
// Key cache for layer 0: [1, total_seq, num_kv_heads, head_dim]
// = [1, total_seq, 8, 128]
let key_cache = TensorRef {
    id: TensorId(30),
    shape: Shape(vec![
        Dim::Static(1),  // Batch = 1 for autoregressive generation
        Dim::Param(ParamDim {
            name: ParamName::TotalSeqLen,
            min: 1,
            max: 8192,
            alignment: 1,
        }),
        Dim::Static(8),   // num_kv_heads
        Dim::Static(128), // head_dim
    ]),
    dtype: DType::F16,
    storage: StorageClass::KVCache {
        layer: 0,
        head_group: 0,
        role: KVRole::Key,
    },
    strides: None,
};
```

### Type Inference Example

```rust
// MatMul: [batch, seq, hidden] @ [hidden, hidden] -> [batch, seq, hidden]
let input = TensorRef {
    id: TensorId(0),
    shape: Shape(vec![
        Dim::Param(ParamDim { name: ParamName::BatchSize, min: 1, max: 32, alignment: 1 }),
        Dim::Param(ParamDim { name: ParamName::SeqLen, min: 1, max: 8192, alignment: 1 }),
        Dim::Static(4096),
    ]),
    dtype: DType::F16,
    storage: StorageClass::Activation { buffer: None },
    strides: None,
};

let weight = TensorRef {
    id: TensorId(1),
    shape: Shape(vec![Dim::Static(4096), Dim::Static(4096)]),
    dtype: DType::F16,
    storage: StorageClass::Weight(WeightBinding {
        name: "weight".to_string(),
        byte_offset: 0,
        byte_size: 4096 * 4096 * 2,
        file_shape: vec![4096, 4096],
        file_dtype: DType::F16,
    }),
    strides: None,
};

// Output has same shape as input (broadcasting over batch and seq dims)
let output = TensorRef {
    id: TensorId(2),
    shape: input.shape.clone(),
    dtype: DType::F16,
    storage: StorageClass::Activation { buffer: None },
    strides: None,
};
```
